# Magic The Gathering Card Finder API

## Descrição

Este projeto é uma API backend desenvolvida em Node.js com Express, responsável por fornecer os recursos necessários para um sistema de busca e gerenciar de cartas do jogo Magic: The Gathering.

A aplicação permite que usuários criem contas, autentiquem-se na plataforma, pesquisem cartas, salvem cartas favoritas e montem seus próprios decks personalizados.

O objetivo do projeto é fornecer uma base sólida e escalável para aplicações web ou mobile que desejem consumir dados relacionados ao jogo e oferecer funcionalidades de organização de decks.

---

## Funcionalidades

- Cadastro de usuários
- Autenticação com login
- Busca de cartas por nome, tipo ou outros critérios
- Visualização de detalhes da carta
- Salvamento de cartas favoritas
- Criação de decks personalizados
- Adição e remoção de cartas em decks
- Listagem de decks do usuário

---

## Tecnologias Utilizadas

- Node.js
- Express
- Banco de dados (a definir ou especificar: MongoDB, PostgreSQL, etc.)
- JWT para autenticação
- Bcrypt para criptografia de senha
